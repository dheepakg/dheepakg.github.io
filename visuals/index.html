Some Charting on Nifty50

Nifty50 on each of it working day

The NSE functions only on weekdays. If we exclude weekends, the number of working days comes down to 261 (365 - 52*2). And then, there comes some more public holidays. This generally varies on each year. The actual working days of the market comes around 250 (+/- 5) days
